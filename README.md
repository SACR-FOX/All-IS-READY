# All IS READY

## 项目简介：

#### 一款面向在校学生党的日常效率类助手App

## 项目功能描述（beta）

- To Do List (每日计划定时提醒 / 短信&通知)

- 课程时间表（智能显示下一节课程信息）

- 文件资料管理 （pdf、office 文件保存与预览）

- 定时器/倒计时 （百分比显示今日学习目标达成进度）

- 学习排行榜

- 班级任务发布

- 学习热力分布图（周围有多少人 在一起学习）

- 学习课程社区
